# VeryGoodCoffee

iOS app to discover and save your favorite coffee pictures.

## Requirements

- iOS 18.0+
- Xcode 16.0+

## How to Run

1. Open `VeryGoodCoffee.xcodeproj` in Xcode
2. Select a simulator or device
3. Press `Cmd + R` to build and run

## How to Use

- Tap the image to load a new coffee picture
- Double-tap on the image or tap the star button to save as favorite
- Switch to the Favorites tab to view saved pictures offline
- Hold on the image to remove it from the favorite pictures

## Architecture

MVVM pattern with SwiftUI, Combine, and SwiftData for local persistence.
